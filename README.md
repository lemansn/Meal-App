# Meals-App

It is a React Native Meal App that shows categories, overview and details screen. Also, we can add meals to the favorites screen by clicking the star icon on the detail screen.


# Technology Stack

* React Native

* React 16.8.3

* Hooks

* Context Api

* React Navigation

  - Stack Navigation

  - Drawer Navigation

* Expo
